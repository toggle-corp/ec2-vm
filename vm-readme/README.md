## Directory structure
```
.
├── proxy -> .dotfiles/proxy/proxy
├── repo
│   │── repo-1
│   └── repo-2
└── services
    │── service-1
    └── service-2
```

Here
- `proxy`
    - Contains configs and containers for vm root level proxy
- `repo`
    - Repo/projects which doesn't have running containers
- `services`
    - Repo/projects which have running containers
